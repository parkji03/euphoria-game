## Setup

Create `/config/local_env.yml` and add Facebook/Google API keys
```javascript
google_api_id: {API_ID}
google_secret: {API_SECRET}
facebook_api_id: {API_ID}
facebook_secret: {API_SECRET}
```
All done. :+1:
